## MySQL 로그인

```Shell
mysql -u root -p
Enter password:
```

## MySQL DB 목록 보기
```
mysql> SHOW databases;
```

## DB 생성
```SQL
CREATE DATABASE [DB_name] default CHARACTER SET UTF8;
```

## CREATE TABLE
```
CREATE TABLE [TABLE_name](
  id INT PRIMARY KEY <AUTO_INCREMENT>,
  name VARCHAR(20) NOT NULL,
  address VARCHAR(32) DEFAULT 'Seoul'
)
```

## TABLE 구조 확인하기
```
DESC [TABLE_name];
```
